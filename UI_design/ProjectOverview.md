Project UI Overview

In this task, I designed the main user interface structure for the ride-sharing web application WayMate. The goal of the design was to create a clear and simple navigation for different types of users (passengers, drivers, and unregistered users).

The following main views/pages were prepared:
- HomepagePassengersView
- HomepageDriversView
- HomepageUnregisteredUser
- MyRidesPage
- AvailableRidesPage
- ChatPage
- RideRequestsPage
- OfferRidePage

These pages cover the core functionality of the application, including browsing available rides, offering rides, managing personal rides, communicating through chat, and handling ride requests.

The UI design and layout prototypes were created in Figma.
