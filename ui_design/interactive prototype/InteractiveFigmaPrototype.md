# Interactive Prototype

The user interface of the application was designed using Figma.  
An interactive prototype was created to demonstrate how users interact with the system before the actual implementation.

The prototype includes the main screens of the ride-sharing application such as:

- homepage with ride search
- ride listing
- ride booking
- user profile
- navigation between pages

Interactive elements were added to simulate real user behaviour.  
Buttons, navigation items, and cards allow the user to move between screens, which helps visualize the overall user experience of the application.

The design follows a clean and modern layout with card-based components and simple navigation to ensure clarity and usability.

The prototype was used to test the flow of the application and to better understand how passengers and drivers interact with the system.
